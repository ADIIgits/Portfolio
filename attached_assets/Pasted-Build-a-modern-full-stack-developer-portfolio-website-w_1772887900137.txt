Build a modern full-stack developer portfolio website with a cinematic Apple/iOS-inspired design language.

Tech stack:

* Next.js (React, App Router)
* TypeScript
* TailwindCSS
* Framer Motion for animations
* Responsive design (mobile-first)
* Dark mode as default

Design philosophy:
The website should feel cinematic, minimal, and premium — inspired by Apple's product pages and iOS UI design. Use soft gradients, glassmorphism, blur effects, smooth transitions, and elegant typography. The site should feel immersive and aesthetic rather than a typical developer portfolio.

Visual style:

* Dark background with subtle gradients
* Frosted glass cards (backdrop blur)
* Smooth parallax scrolling
* Large cinematic hero section
* Elegant typography similar to Apple design language
* Minimal color palette (black, charcoal, soft blue accents)
* Smooth hover and scroll animations
* Micro-interactions

IMPORTANT DATA STRUCTURE REQUIREMENT:

All portfolio content must be stored in structured TypeScript objects instead of being hardcoded directly inside components.

Create a dedicated data layer (for example /data folder) containing object-based data models.

Use object-based structures so data can be accessed like:

project.title
project.description
project.techStack
experience.company
experience.role

Example approach:

Create structured objects or interfaces such as:

Profile object
Projects array of objects
Technologies array
Experience array

Example project structure conceptually:

project = {
title,
description,
techStack,
githubLink,
liveLink,
image,
featured
}

Components should dynamically render content by mapping over these objects.

Example:
Projects section should map through the projects array and display project.title, project.description, etc.

This ensures that later I only need to modify the data objects to update the portfolio instead of editing UI code.

Code quality:

* Clean component structure

The portfolio should feel like a premium Apple-style cinematic experience while maintaining a clean object-based architecture where all portfolio content can be updated by modifying structured data objects.

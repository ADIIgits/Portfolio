Update the hero section background effect.

IMPORTANT:
Do NOT change the hero layout, typography, spacing, or positioning of the content. Keep the text ("Aditya Sharma", subtitle, buttons) exactly where it is.

The current hero uses a **large smoky gradient fade** between the solid background and the portrait image. This takes up too much space.

Refactor the effect to use **blur instead of smoke/fade**, similar to the Apple TV / Apple Music hero design.

Goals:
• Reduce the smoky gradient area
• Make the image visible earlier
• Use blur to soften the transition instead of a long gradient

Implementation requirements:

1. Reduce Gradient Width

Currently the gradient fades too slowly across the hero.

Replace the large gradient with a **shorter gradient mask** so the image begins appearing earlier.

Example concept:

background-image:
linear-gradient(
to right,
rgba(255,255,255,0.95) 0%,
rgba(255,255,255,0.85) 20%,
rgba(255,255,255,0.35) 40%,
rgba(255,255,255,0.0) 55%
),
url(heroImageUrl)

The transition should complete around **50–60% width**, not across the whole hero.

2. Add Blur Layer

Instead of a long fade, add **blur to the image behind the text area**.

Use CSS blur or backdrop-filter so the image softly blends into the background.

Example concept:

filter: blur(8px)
or
backdrop-filter: blur(12px)

The blur should only affect the **left side where text sits**, while the portrait area remains sharp.

3. Create a Masked Blur Transition

The blur should gradually reduce as the image moves toward the right side.

Left side:
more blur

Center:
light blur

Right side:
no blur (sharp portrait)

This creates a cinematic depth effect.

4. Preserve Portrait Visibility

The developer portrait on the right side must remain:

• sharp
• high contrast
• clearly visible

Only the text side should be softened.

5. Maintain the Cinematic Look

Keep the design premium and minimal:

• no heavy fog
• no thick gradient
• clean blur transition

The final effect should resemble **Apple TV hero sections**, where the image is visible but softly blurred behind UI elements.

Goal:
Create a cleaner cinematic hero where the portrait appears sooner, the transition is driven by blur instead of smoke, and the design feels more modern and Apple-like.
